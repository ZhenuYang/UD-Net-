# UD-Net：基于自适应标记分布生成的人脸抑郁识别研究

## GitHub 项目信息建议

- 中文项目名：`UD-Net：基于自适应标记分布生成的人脸抑郁识别研究`
- 英文仓库名：`UD-Net-Depression-Recognition`
- GitHub 简介：`面向抑郁症客观辅助评估的轻量级人脸抑郁识别框架，结合标记分布学习、误差驱动方差校正与小样本轻量化建模。`

## 项目简介

本项目来源于“基于自适应标记分布生成的人脸抑郁识别研究”结题工作，核心方法为 `UD-Net`，即一种**不确定性引导的轻量级抑郁识别框架**。项目聚焦抑郁症客观辅助评估场景，尝试利用面部视觉信号实现对抑郁程度的智能化量化分析。

围绕真实场景中的三类关键问题：

- 主观量表带来的标签模糊性
- 复杂视频样本带来的数据异方差性
- 公开数据规模有限导致的小样本过拟合

本项目以标记分布学习（LDL）为理论基础，提出了面向不确定性建模的 `UD-Net` 框架，并从算法设计、轻量化建模、数据预处理、数据增强和多模态拓展等方面进行了系统研究。

## 论文对应的核心思路

`UD-Net` 的核心不是简单地回归一个抑郁分值，而是让模型学习一个更符合真实临床标注特性的概率分布表示。论文中的主要思路包括：

- 以标记分布学习替代单值回归，更好地表达抑郁状态的连续性与不确定性
- 设计误差驱动的方差校正策略，根据预测误差动态调整分布形态
- 使用轻量级 `ResNet-18` 作为骨干网络，在保证效率的同时提升可部署性
- 引入部分冻结策略，缓解小样本条件下深层网络过拟合
- 构建包含 `MTCNN` 人脸对齐与统一采样的视频预处理流水线
- 探索基于扩散模型的数据增强，以及沉浸式 `VR` 环境下的生理-行为多模态扩展

## 论文中的实验结果

根据结题报告与论文内容，`UD-Net` 在 `AVEC 2014` 数据集上取得了如下表现：

- `MAE = 6.88`
- `RMSE = 8.86`
- `CCC = 0.5709`
- 计算开销约为 `1.8 GFLOPs`

该结果体现了 `UD-Net` 在识别精度与部署效率之间较好的平衡，也说明轻量级 2D 网络在不确定性建模充分时，可以具备与复杂时空模型竞争的能力。

## 仓库内容说明

当前仓库主要整理的是项目的软件原型与核心实现骨架，便于做项目展示、成果归档与后续扩展，目录中的关键文件包括：

```text
UD-Net_Software/
├─ app.py               # Gradio 界面入口
├─ preprocess.py        # 视频预处理、人脸检测与关键帧提取
├─ model.py             # 基于 ResNet-18 的轻量化网络结构定义
├─ requirements.txt     # 运行依赖
├─ README.md            # 项目说明
└─ udnet_best.pth       # 可选模型权重文件
```

其中：

- `app.py` 提供了项目软件界面与演示流程
- `preprocess.py` 对应论文中的视频预处理部分，包含 `MTCNN` 人脸检测与基础采样逻辑
- `model.py` 保留了轻量级双头结构的实现思路，包含均值分支与方差分支



## 运行方式

```bash
pip install -r requirements.txt
python app.py
```

运行后可在浏览器中打开本地界面，上传访谈视频并查看预处理与演示结果。

## 使用说明

1. 启动程序并打开 Gradio 页面。
2. 上传待分析的访谈视频文件。
3. 系统执行关键帧抽取与人脸检测。
4. 页面展示人脸结果与演示版分析输出。
